Утилита GMXCSA от izzx

2020.09.17. v1.0
GMXCSA - утилита под Win для конвертации подготовленных файлов *.cel в формат экрана Scorpion GMX.
Итоговые файлы можно посмотреть на Scorpion GMX с помощью GMXPV v1.0
gmxcsa8.exe - конвертирует в 8 цветов
gmxcsa15.exe - конвертирует в 15 цветов (с повышенной яркостью). Результат, на мой взляд, хуже, но для некоторых изображений подойдёт.

Для работы утилиты нужен Net Framework 4.x
Проверена на Win 7 32/64.
Файлы *cel в строго определённом формате нужно готовить в граф. редакторе GIMP.
Запуск утилиты в командной строке: gmxsca8 <имя файла без расширения>
Например, есть файл "alissa.cel", запускаем: "gmxsca8 c:\123\alissa".
В той же папке, где лежал файл .cel, появятся файлы 
alissa-a.C (16000 байт, атрибуты в формате экрана GMX)
alissa-p.C (16000 байт, сама картинка)
alissa.C (32768 байт, итоговый файл)

Последный файл скопировать на ZX любым способом.

Как конвертировать в GIMP.
Сначала положить палитру "Без-имени-ZX.gpl" (прилагается) в папку %user%\gimp\pallets
или же импортировать в диалоге цветовых образцов.
Открыть нужную картинку.
Сделать размер картинки 320 пикселей ширина 200 высота с помощью инструментов 
Изображение - Размер изображения (скрепки сцепить чтобы не исказить пропорции)
и Изображение - Размер холста
Затем Изображение - скрепки расцепить- установить размер 640*200
Картинка будет вытянута по горизонтали в 2 раза. 
Так как на GMX пиксели не квадратные, при просмотре получится то, что надо, но слегка уже по вертикали на 7-8%.
Затем Изображение - Режим Индексированный
Выбрать Использовать заказную палитру
Выбрать нашу палитру Без имени ZX (16)
Размывание цвета по Флойду - Преобразовать
Затем вернуть Изображение - Режим RGB
Файл - Экспортировать как - формат KISS CEL (*.cel), расширение поправить тоже на cel
Экспортировать.
Должен получиться файл ровно 512032 байт (32 байт заголовок).
Внутри варианты цветов только такие (четвёртый байт игнорируется):

#000000

#0000c0

#c00000

#c000c0

#00c000

#00c0c0

#c0c000

#c0c0c0

#000000

#0000ff

#ff0000

#ff00ff

#00ff00

#00ffff

#ffff00

#ffffff

Начало заголовка
4B 69 53 53 20 20 00 00 80 02 C8

Прилагается пример такого файла "alissa.cel", полученного из картинки "alissa.jpg"
